# UC028 - 관리자 관리

최고관리자가 관리자를 등록, 조회, 수정, 삭제하는 유스케이스

## 주 액터(Primarty Actor)
최고관리자

## 보조 액터(Secondary Actor)

## 사전 조건(Preconditions)
- 최고관리자 계정으로 로그인한 상태이다.

## 종료 조건(Postconditions)



## 시나리오(Flow of Events)
### 관리자 조회
1. 액터는 관리자 페이지의 관리자 관리를 클릭한다.
2. 시스템은 관리자 목록을 출력한다. 


### 관리자 등록
1. 액터는 관리자 페이지의 관리자 등록/수정 버튼을 누른다. 
2. 시스템은 기존 관리자 목록과 회원 목록을 출력한다. 
3. 액터가 회원 목록 중 하나를 선택해 등록 버튼을 누른다.
4. 시스템은 권한 설정 창을 출력한다.
5. 액터가 권한 설정을 마치고 확인 버튼을 누른다.
6. 시스템은 '등록하시겠습니까?' 확인하는 내용을 출력한다.
  - 확인 버튼을 누르면, 시스템은 변경사항을 저장하고 '관리자 등록' 유스케이스의 2번으로 간다.
  - 취소 버튼을 누르면 확인 창이 닫힌다.

### 관리자 수정
1. 액터는 관리자 페이지의 관리자 등록/수정 버튼을 누른다.
2. 시스템은 기존 관리자 목록과 회원 목록을 출력한다.
3. 액터가 기존 관리자 목록에서 하나를 선택해 수정 버튼을 누른다.
4. 시스템은 권한 설정 창을 출력한다.
5. 액터가 권한 설정을 마치고 확인 버튼을 누른다.
6. 시스템은 '수정하시겠습니까?' 확인하는 내용을 출력한다.
  - 확인 버튼을 누르면, 시스템은 변경사항을 저장하고 '관리자 등록' 유스케이스의 2번으로 간다.
  - 취소 버튼을 누르면 확인 창이 닫힌다.


### 관리자 삭제
1. 액터는 관리자 페이지의 관리자 관리를 클릭한다. 
2. 시스템은 관리자 목록을 출력한다.
3. 삭제하려는 관리자를 선택하고 삭제 버튼을 클릭한다.
2. 시스템은 경고 창('정말 삭제하시겠습니까?')을 출력한다.
  - 확인 버튼을 누르면 시스템은 해당 관리자룰 삭제한 후 '관리자 조회' 유스케이스의 2번으로 간다.
  - 취소 버튼을 누르면 시스템은 해당 창을 닫는다.
